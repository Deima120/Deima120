## 1. Crear el README de perfil (lo más importante)

GitHub muestra un README especial en la parte superior de tu perfil si creas un repo con **el mismo nombre que tu usuario**.

### Pasos

1. Ve a [github.com/new](https://github.com/new)
2. Nombre del repositorio: **`Deima120`** (exactamente igual a tu usuario)
3. Marca **Public**
4. Marca **Add a README file**
5. Clic en **Create repository**
6. Reemplaza el contenido del README con el archivo `README.md` de esta carpeta
7. Personaliza:
   - Tu LinkedIn (`TU_USUARIO`)
   - Tu email
   - Tu ciudad/país en el bloque `const emanuel`
   - Descripciones de proyectos si cambian
